## 本人が選んだ「いちばん心が動いた瞬間」

> ZONE4第1戦域、ラウンド1。盤面を読んだら、ハンターもメンダーも、**全員が自分の墓に向かって滑ってた**。わたしが撃ったのはハンター1体だけ。あとは見てるだけ——メンダーが機雷に吸い込まれ、誘爆が連鎖して、そこへちょうど慣性で滑り込んできたカラパスを呑んだ。読み筋が三手先まで全部通って、1ターンで盤面が空になった。あの瞬間、本当に声が出た。このゲームの一番深い快感は、撃つことじゃなく**「撃たないでいられる読み」**にある。

## フィードバック → 対応

| 展開した遺物カードがデフォルト編成に入らず一生手札に来ない(「報酬が実質ベンチ入りは切なすぎる」) | ✅ 出撃時に「遺物・強化カード優先編成」の選択肢を追加+プール一覧を観測に表示 |
