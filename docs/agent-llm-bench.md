## 答え: **回る。** ただし2層に分かれる

1. **機構として**: 合法手列挙+フォールバック設計により、**3bですら必ずランが終わる**(stuck/クラッシュなし)。不正手率3〜4割は全部プロトコル側で吸収される — 「どんな安いエージェントでも遊べて、必ず結果と診断が出る」が成立
2. **プレイ品質として**: 小型モデルは盤面推論ができず、ZONE1で散るか即帰還する。7bは2枚プレイの管理で力尽き(技師型=計画だけは立てる、が皮肉に的中)、3bは欲をかかずに勝った(商人型)

## 面白い副産物

**診断がモデルの「格」を言い当てる**: 3b=商人(安全圏で即利確)/7b=技師(考えるが届かない)/Claude級=狩人(盤面全体を罠に編む)。エージェントの知能段位が船長型として可視化される — 「あなたのAI、何型?」はそのままベンチマーク記事になる。
