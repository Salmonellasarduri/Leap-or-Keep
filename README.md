# Leap or Keep — One more leap.

宇宙版カード型ローグライク。死んだ文明の遺物を回収する深宇宙サルベージャーとなり、「もう1ゾーン深く跳ぶか、戦利品を持って帰るか」を選び続ける。Gloomhavenリスペクトのカード消耗システム(カード=行動=寿命)+ Into the Breach式の敵行動予告 + 宇宙ならではの慣性(ドリフト)。

## 遊び方

**`index.html` をダブルクリックするだけ**(ビルド・ネットワーク不要)。

- URLに `?seed=数字` を付けると同じ展開を再現できる(例: `index.html?seed=42`)

## ルール3行

1. 毎ターン手札から2枚選び、一方の**上半分(航行)**+他方の**下半分(戦闘)**を旗艦か随伴機に割り当てる(順番・割当は自由)。手札が尽きたら強制休息=**1枚永久ロスト**。全カードを失うか旗艦が壊れたら敗北。
2. 移動したユニットは次ラウンド**同方向に1マス滑る**(慣性ドリフト — 半透明の影が滑り先)。直進はタダ、止まるのは有料。敵もすべる。端はループする。
3. ゾーンをクリアするたび遺物を**封印**(持ち帰り価値)か**展開**(今すぐ強カード、ただし失うかも)し、**跳ぶ**(カード1枚を燃料に)か**帰る**(勝利確定)かを選ぶ。

操作のコツ: カード・ユニット・赤マスにマウスを乗せると全部説明が出る。攻撃対象にホバーでダメージ予測。**「敵ターンへ」を押すまではターン丸ごとやり直せる(⟲)**ので、怖がらず試そう。

## 開発

- 仕様: [SPEC.md](SPEC.md)(デザインピラー・UX強調文法・ジレンマ監査表含む)
- テスト: `npm test` — ルール検証+全カード網羅スモーク+3方策収支シミュレーション(計109件)
- スクリーンショット検証: `npm run shot`(Playwright) — 全画面シナリオを tmp/shot-*.png に撮影
- ローカルサーバー(任意): `npm run serve` → http://localhost:8322/
- バランス定数は `index.html` 冒頭の `CONFIG` と各定義表(`CARD_DEFS` / `SHIP_DEFS` / `ENEMY_DEFS` / `ZONE_TABLE`)に集約
- 初回のみ: `npm i` と `npx playwright install chromium`(スクショ用)

## v0.6.0-alpha の追加(Steam売れ筋比較による拡張 — docs/steam-self-review.md)

- **船体3クラス**: ヴァグランツ(標準) / ベリーロール(物理無傷・与ダメ+1、₢15) / アストラ(射程+1・HP5、₢30) — 専用デッキ+パッシブ
- **カード強化(+)**: 各ゾーン第1戦域クリアで1枚強化(主要数値+1)
- **アセンション4段**: 生還するたび次の深淵が解放(敵HP+1 → 燃料2枚 → 全敵攻撃+1)
- **敗北ドリップ**: 負けても回収クレジット(失敗ランを無駄にしない)
- **ランスコア+自己ベスト**(物理キルはスコア加点+連鎖スタンプで祝福)

## v0.5.0-alpha の主な要素

- 5×5トーラス盤+ハザード(岩塊/係留機雷=誘爆連鎖)+回収コンテナ
- 敵5種(同士討ちあり — クラゲの触手や機雷に敵を押し込め)
- カード21種(複合: 攻撃+引き/攻撃+突き飛ばし、🔥ロストアクション: 寿命と引き換えの切り札)
- メタ進行: クレジット→格納庫でカード解放、遺物インストール、出撃デッキ編成(localStorage)
- ゾーンイベント、難易度「深淵」、合成SE、俯瞰チルト、キル演出
