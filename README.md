# Leap or Keep — One more leap.

**▶ いますぐ遊ぶ: https://salmonellasarduri.github.io/Leap-or-Keep/** (ブラウザだけ・無料・スマホ可)

宇宙版カード型ローグライク。死んだ文明の遺物を回収する深宇宙サルベージャーとなり、「もう1ゾーン深く跳ぶか、戦利品を持って帰るか」を選び続ける。Gloomhavenリスペクトのカード消耗システム(カード=行動=寿命)+ Into the Breach式の敵行動予告 + 宇宙ならではの慣性(ドリフト)。

ラン終了時には**船長診断**(全8型)があんたのプレイスタイルを言い当てる。『海賊型 — ぶつかってから考える人』。結果はXにポストできる。 #LeapOrKeep

## ルール3行

1. 毎ターン手札から2枚選び、一方の**上半分(航行)**+他方の**下半分(戦闘)**を旗艦か随伴機に割り当てる(順番・割当は自由)。手札が尽きたら強制休息=**1枚永久ロスト**。全カードを失うか旗艦が壊れたら敗北。
2. 移動したユニットは次ラウンド**同方向に1マス滑る**(慣性ドリフト — 半透明の影が滑り先)。直進はタダ、止まるのは有料。敵もすべる。端はループする。
3. ゾーンをクリアするたび遺物を**封印**(持ち帰り価値)か**展開**(今すぐ強カード、ただし失うかも)し、**跳ぶ**(カード1枚を燃料に)か**帰る**(勝利確定)かを選ぶ。

操作のコツ: カード・ユニット・赤マスにマウスを乗せると全部説明が出る。**「敵ターンへ」を押すまではターン丸ごとやり直せる(⟲)**。初めてなら📚訓練航行(3分)から。

## 主な要素(v0.9)

- 5×5トーラス盤+ハザード(岩/機雷誘爆)+重力渦+太陽フレア
- 敵11種(各敵=攻略法の語彙: 装甲種は物理でしか割れない、修復網は最優先で潰せ…)+**ラスボス3種からランごとに抽選**
- カード25種(ロストアクション=寿命と引き換えの切り札)・船体3クラス・カード強化・アセンション4段
- **航行契約**: リスクを自選して盛るほどスコア倍率と称号(アークナイツ危機契約リスペクト)
- **船長診断**: プレイの選択(物理/危険/熟慮)から8型+サブ称号を判定 → Xポスト/カード画像保存
- メタ進行: クレジット/格納庫/遺物インストール/戦績トップ5/型図鑑(localStorage、サーバーなし)

## ローカルで遊ぶ・開発する

`index.html` をダブルクリックするだけ(ビルド・ネットワーク不要)。`?seed=数字` で同じ展開を再現。

- 仕様: [SPEC.md](SPEC.md)(デザインピラー・UX強調文法・ジレンマ監査表・調査付録)
- テスト: `npm test` — ルール検証+全カード網羅スモーク+3方策収支シミュレーション(計258件)
- スクリーンショット検証: `npm run shot`(Playwright)/ 自動プレイ: `node tools/playrun.mjs`
- バランス定数は `index.html` の `CONFIG` と各定義表(`CARD_DEFS`/`SHIP_DEFS`/`ENEMY_DEFS`/`ZONE_TABLE`/`CONTRACT_DEFS`/`CAPTAIN_TYPES`)に集約
- 初回のみ: `npm i` と `npx playwright install chromium`(検証ツール用)
